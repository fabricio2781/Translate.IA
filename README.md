# Translatebot.IA
 O Translatebot foi criado com o objetivo de ser um chat inteligente que possa tirar todas as duvidas dos usuarios como se fosse um professor poliglota de idiomas.
 
Passo 1
Executar terminal (CMD ou PowerShell)

passo 2
selecionar diretorio do chat (pasta onde foi instalado)
por exemplo (cd "local instalado")

passo 3
Instalar Bibliotecas: pip install flask groq python-dotenv

passo 4
Para o Chat funcionar o usuario deve gerar uma API no site https://groq.com e abir o diretorio do chat e Colocar API Key gerada para funcionar

passo 5
executar: python main.py
Rodar no navegador: http://127.0.0.1:5000
